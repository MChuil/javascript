Promesas en JavaScript

Una promesa es un objeto que producirá un valor único en algún momento en el futuro. Si la promesa tiene exito, producira el valor resuelto, pero si algo sale mal, generar una razon por la cual la promesa fallo.

Sintaxis: function promesa(){ const promise = new Promise((resolve, reject) => { const num = 2 if(num % 2 ==0){ resolve('Es numero par') }else{ reject() } }) return promise }

¿Que es la asincronia en JavaScript?

Es una técnica que permite al script inicar una tarea de larga duración y seguir respondiendo a otros eventos mientras esa tarea es ejecutada, en lugar de tener que esperar hasta que esa tarea haya terminado.
