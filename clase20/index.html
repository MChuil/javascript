Petición HTTP

Fetch API: Proporciona una interfaz JavaScript para acceder y manipular parte del canal HTTP, peticiones y respuestas.

Esta basado en promesas, por lo cual tiene un 'resolve' y un 'reject'. El 'Response', tiene varios métodos:

  • arrayBufer()
  • blob()
  • clone()
  • formData()
  • json()
  • text()
